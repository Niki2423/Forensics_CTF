# Digital Forensics CTF Challenges

This repository contains two digital forensics Capture The Flag (CTF) challenges:

1. **Image Metadata Disk Forensics Challenge**
2. **Suspicious Network Traffic PCAP Challenge**

These challenges are designed for learning and practicing digital forensics, cyber forensics, and incident investigation.
