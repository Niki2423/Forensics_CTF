# Solution — Image Metadata Disk Forensics Challenge

## Challenge Name
**Deleted Cat Evidence**

---

## Overview

This challenge simulates a small digital forensics investigation involving a deleted image file stored inside a forensic disk image. The hidden flag is embedded in the image metadata, so the player must recover the deleted image and inspect its metadata to extract the flag.

This challenge is related to **disk forensics**, **file carving**, and **metadata analysis**.

---

## Objective

Recover the deleted image from the provided disk image and extract the hidden flag from the recovered file.
